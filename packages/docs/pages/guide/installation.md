# 安装

## 环境要求

- Vue 3.5+
- Element Plus 2.11+
- TypeScript 5.9+ (推荐)

## 安装方式

::: code-group

```sh [npm]
npm install @yetuzi/vue3-query-components
```

```sh [yarn]
yarn add @yetuzi/vue3-query-components
```

```sh  [pnpm]
pnpm add @yetuzi/vue3-query-components
```

:::


## 浏览器兼容性

- 现代浏览器，支持 ES2018+ 和 ES Modules
- 不支持 IE 浏览器
