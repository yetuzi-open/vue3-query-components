# 基础组件

这里汇总展示 `packages/components/src/components` 下除配置组件和核心组件之外的基础组件。

当前基础组件列表：

| 组件名 | 说明 |
| --- | --- |
| `CommonButton` | 按钮组件 |
| `CommonInput` | 输入框组件 |
| `CommonInputNumber` | 数字输入框组件 |
| `CommonSelect` | 下拉选择组件 |
| `CommonDatePicker` | 日期选择器组件 |
| `CommonTimePicker` | 时间选择器组件 |
| `CommonCascader` | 级联选择组件 |
| `CommonTreeSelect` | 树选择组件 |
| `CommonRadio` | 单选框组件 |
| `CommonCheckbox` | 复选框组件 |
| `CommonSwitch` | 开关组件 |
| `CommonUpload` | 上传组件 |

这些组件主要作为 `CommonForm`、`CommonDialog` 等上层业务组件的基础控件使用，也可以单独在业务页面中直接使用。

## 分类说明

### 输入类

- `CommonInput`
- `CommonInputNumber`

### 选择类

- `CommonSelect`
- `CommonDatePicker`
- `CommonTimePicker`
- `CommonCascader`
- `CommonTreeSelect`
- `CommonRadio`
- `CommonCheckbox`
- `CommonSwitch`

### 交互类

- `CommonButton`
- `CommonUpload`
